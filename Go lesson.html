شرح دورة أساسيات Go

لغة Go هي لغة برمجة مفتوحة المصدر من تطوير شركة Google، وتتميز بالبساطة والأداء العالي.

الدرس الأول: تثبيت Go

يمكنك تحميل وتثبيت Go من الموقع الرسمي: https://go.dev/dl/

شرح مفصل:
  • 1. مقدمة عن Go: Go (أو Golang) هي لغة برمجة مفتوحة المصدر تم تطويرها بواسطة شركة Google في عام 2009. تتميز Go بالبساطة، السرعة، والكفاءة، وهي مصممة لمعالجة التطبيقات المتوازية (Concurrency) بكفاءة عالية.
  • 2. تثبيت Go: يمكنك تثبيت Go على نظامك عن طريق تحميل الإصدار المناسب من الموقع الرسمي.
مثال:
package main import "fmt" func main() { fmt.Println("Hello, World!") }

الدرس الثاني: أول برنامج بلغة Go

هذا مثال لبرنامج "Hello, World!" بلغة Go:

شرح مفصل:
  • 1. إنشاء أول برنامج: يمكنك إنشاء أول برنامج لك بلغة Go باستخدام دالة `main`.
مثال:
package main import "fmt" func main() { fmt.Println("Hello, World!") }

الدرس الثالث: المتغيرات في Go

يمكنك تعريف المتغيرات باستخدام الكلمة المفتاحية `var` أو الاختصار `:=`:

شرح مفصل:
  • 1. تعريف المتغيرات: يمكنك تعريف المتغيرات باستخدام `var` مع تحديد النوع أو بدون تحديد النوع.
  • 2. الاختصار `:=`: يمكنك استخدام `:=` لتعريف المتغيرات بدون الحاجة إلى تحديد النوع.
مثال:
package main import "fmt" func main() { var name string = "Ahmed" var age int = 25 fmt.Println("الاسم:", name) fmt.Println("العمر:", age) }
package main import "fmt" func main() { city := "Cairo" population := 20000000 fmt.Println("المدينة:", city) fmt.Println("عدد السكان:", population) }

الدرس الرابع: الحلقات والتكرار

تستخدم Go الحلقة `for` لتنفيذ التكرار:

شرح مفصل:
  • 1. الحلقة `for` الأساسية: يمكن استخدام `for` بنفس الطريقة التي نستخدم بها `for` في لغات أخرى مثل C أو Java.
  • 2. استخدام `for` كـ `while`: لا تحتوي Go على `while`، لكن يمكننا استخدامها بنفس الفكرة من خلال `for` بدون تعريف عدّاد.
مثال:
package main import "fmt" func main() { for i := 0; i < 5; i++ { fmt.Println("التكرار رقم:", i) } }
package main import "fmt" func main() { i := 0 for i < 5 { fmt.Println("i =", i) i++ } }

الدرس الخامس: الدوال في Go

تعريف واستدعاء دالة:

شرح مفصل:
  • 1. تعريف دالة بسيطة واستدعاؤها: يمكنك تعريف دالة بسيطة واستدعاؤها باستخدام اسمها.
  • 2. دالة تحتوي على معاملات (Parameters): يمكنك تعريف دالة تستقبل معاملات.
  • 3. دالة تُرجع قيمة (Return Value): يمكنك تعريف دالة تُرجع قيمة.
مثال:
func greet(name string) { fmt.Println("مرحبًا", name, "😊") } func main() { greet("أحمد") // استدعاء الدالة وتمرير "أحمد" greet("سارة") // استدعاء الدالة وتمرير "سارة" }
func sum(a int, b int) int { return a + b } func main() { result := sum(10, 5) // استدعاء الدالة وتخزين النتيجة fmt.Println("المجموع:", result) }

الدرس السادس: الجمل الشرطية

الجمل الشرطية تسمح باتخاذ القرارات بناءً على شروط معينة.

شرح مفصل:
  • 1. الجملة الشرطية `if`: يمكنك استخدام `if` لتنفيذ كود بناءً على شرط معين.
  • 2. الجملة الشرطية `else`: يمكنك استخدام `else` لتنفيذ كود إذا لم يتحقق الشرط.
مثال:
package main import "fmt" func main() { age := 18 if age >= 18 { fmt.Println("أنت بالغ.") } else { fmt.Println("أنت قاصر.") } }

الدرس السابع: المصفوفات (Arrays)

المصفوفات هي هياكل بيانات تخزن مجموعة من العناصر في ترتيب معين.

شرح مفصل:
  • 1. إنشاء المصفوفات: يمكنك إنشاء مصفوفة باستخدام الأقواس المربعة `[]`.
  • 2. الوصول إلى العناصر: يمكنك الوصول إلى العناصر باستخدام الفهرس.
مثال:
package main import "fmt" func main() { var numbers = [5]int{1, 2, 3, 4, 5} fmt.Println(numbers[0]) // ✅ 1 }

الدرس الثامن: الشرائح (Slices)

الشرائح هي هياكل بيانات مرنة تُستخدم لتخزين مجموعة من العناصر.

شرح مفصل:
  • 1. إنشاء الشرائح: يمكنك إنشاء شريحة باستخدام `make`.
  • 2. إضافة العناصر إلى الشرائح: يمكنك إضافة عناصر إلى الشريحة باستخدام `append`.
مثال:
package main import "fmt" func main() { var numbers = []int{1, 2, 3, 4, 5} numbers = append(numbers, 6) fmt.Println(numbers) // ✅ [1 2 3 4 5 6] }

الدرس التاسع: الخرائط (Maps)

الخرائط هي هياكل بيانات تخزن البيانات في أزواج مفتاح-قيمة.

شرح مفصل:
  • 1. إنشاء الخرائط: يمكنك إنشاء خريطة باستخدام `make`.
  • 2. الوصول إلى القيم: يمكنك الوصول إلى القيم باستخدام المفتاح.
مثال:
package main import "fmt" func main() { var person = map[string]string{ "name": "علي", "age": "25", } fmt.Println(person["name"]) // ✅ علي }

الدرس العاشر: الاستثناءات (Exceptions)

الاستثناءات هي أخطاء تحدث أثناء تنفيذ البرنامج. يمكن التعامل معها باستخدام `panic` و `recover`.

شرح مفصل:
  • 1. التعامل مع الاستثناءات: يمكنك استخدام `panic` لإيقاف البرنامج عند حدوث خطأ.
  • 2. استعادة الاستثناءات: يمكنك استخدام `recover` لاستعادة البرنامج من حالة `panic`.
مثال:
package main import "fmt" func main() { defer func() { if r := recover(); r != nil { fmt.Println("تم استعادة البرنامج من حالة panic:", r) } }() panic("حدث خطأ ما!") }
