اساسيات البرمجه

تعلم الأساسيات التي تحتاجها لبدء رحلتك في عالم البرمجة.

HTML

اكتشف كيفية بناء الهيكل الأساسي لصفحات الويب باستخدام HTML.

CSS

تعلم كيفية تنسيق وتزيين صفحات الويب باستخدام CSS.

JavaScript

أضف التفاعل والديناميكية إلى مواقعك باستخدام JavaScript.

Typescript

تعلم كيفية تحسين جودة وكفاءة كود JavaScript باستخدام TypeScript.

EJS

استخدم EJS لإنشاء قوالب ديناميكية لصفحات الويب.

#C

تعلم لغة C# القوية لبناء تطبيقات سطح المكتب والألعاب.

Python

ابدأ رحلتك في البرمجة بلغة Python البسيطة والقوية.

Java

تعلم Java لبناء تطبيقات قوية ومتعددة الاستخدامات.

problemsolving

طور مهاراتك في حل المشكلات البرمجية باستخدام تقنيات فعالة.

هدف الموقع
