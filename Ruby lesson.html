شرح دورة أساسيات Ruby

Ruby هي لغة برمجة كائنية التوجه (OOP) تُستخدم في تطوير الويب، وأتمتة المهام، ومعالجة البيانات، وأكثر! 🚀

الدرس الأول: أساسيات Ruby

Ruby هي لغة برمجة كائنية التوجه (OOP) تُستخدم في تطوير الويب، وأتمتة المهام، ومعالجة البيانات، وأكثر! 🚀

شرح مفصل:
  • تشغيل Ruby: يمكنك تثبيت Ruby من الموقع الرسمي.
  • المتغيرات: Ruby لا تحتاج إلى تصريح نوع المتغير، فهو ديناميكي.
مثال:
ruby -v
name = "Ruby" age = 25 price = 19.99 is_active = true puts name # ✅ Ruby puts age # ✅ 25

الدرس الثاني: المحددات في Ruby

المحددات (Modifiers) في Ruby هي كلمات مفتاحية تتحكم في الوصول إلى البيانات والدوال داخل الكلاس. تساعد هذه المحددات في تنظيم الكود وحماية البيانات وفقًا لمبادئ البرمجة الكائنية (OOP).

شرح مفصل:
  • public: يسمح بالوصول إلى الدالة أو المتغير من أي مكان داخل الكود. كل الدوال افتراضيًا تكون public إلا إذا تم تغيير ذلك.
  • private: الدوال الخاصة لا يمكن استدعاؤها من خارج الكلاس. يمكن استدعاؤها فقط من داخل نفس الكلاس.
مثال:
class Person def say_hello puts "مرحبًا بالجميع! 😀" end end person = Person.new person.say_hello # ✅ مرحبًا بالجميع! 😀
class Person def initialize(name) @name = name end def show_name secret_name end private # 👈 كل ما بعد هذه الكلمة يصبح خاصًا def secret_name puts "اسمي سري: #{@name} 🤫" end end person = Person.new("علي") person.show_name # ✅ اسمي سري: علي 🤫 person.secret_name # ❌ خطأ! لا يمكن استدعاؤها من الخارج

الدرس الثالث: التصفح في Ruby

Ruby توفر مكتبات وأدوات قوية للتعامل مع الملفات والمجلدات، مما يسمح بإنشاء الملفات، قراءتها، التعديل عليها، وإدارتها بسهولة.

شرح مفصل:
  • الحصول على المسار الحالي: للحصول على المسار الحالي الذي يعمل فيه البرنامج.
  • تغيير الدليل: لتغيير المجلد الحالي إلى مجلد آخر.
مثال:
puts Dir.pwd
Dir.chdir("/path/to/directory") # استبدل بالمسار المطلوب puts Dir.pwd # ✅ يعرض المسار الجديد

الدرس الرابع: الأنماط المتقدمة

في Ruby، توجد ميزات متقدمة تساعد في تحسين الأداء، وتنظيم الكود، وتوفير إمكانيات برمجية قوية. هنا سنستعرض أهم الأنماط المتقدمة مع أمثلة عملية.

شرح مفصل:
  • الميتا برمجة: الميتابرمجة تعني كتابة كود قادر على تعديل نفسه أثناء التشغيل، وهو مفهوم قوي جدًا في Ruby.
  • الموديولات (Modules) والمزج (Mixin): الموديولات توفر طريقة لإضافة وظائف إضافية إلى الكلاسات بدون وراثة مباشرة.
مثال:
class Person def greet puts "مرحبًا!" end end person = Person.new person.send(:greet) # ✅ يستدعي الدالة greet بدون مناداتها مباشرةً
module Speak def say_hello puts "مرحبًا! 🎉" end end class Human include Speak # إدراج الموديول داخل الكلاس end person = Human.new person.say_hello # ✅ مرحبًا! 🎉

الدرس الخامس: التعامل مع الملفات

Ruby توفر أدوات قوية للتعامل مع الملفات، مما يسمح بقراءة وكتابة الملفات بسهولة.

شرح مفصل:
  • قراءة الملفات: يمكن قراءة محتوى الملف باستخدام `File.read`.
  • كتابة الملفات: يمكن كتابة محتوى إلى ملف باستخدام `File.write`.
مثال:
content = File.read("example.txt") puts content
File.write("example.txt", "Hello, Ruby!")

الدرس السادس: الجمل الشرطية والحلقات

الجمل الشرطية والحلقات هي أدوات أساسية في البرمجة تسمح باتخاذ القرارات وتكرار العمليات.

شرح مفصل:
  • الجمل الشرطية: مثل `if`، `else`، و `elsif`.
  • الحلقات: مثل `for`، `while`، و `each`.
مثال:
age = 18 if age >= 18 puts "أنت بالغ." else puts "أنت قاصر." end
numbers = [1, 2, 3, 4, 5] numbers.each do |number| puts number end

الدرس السابع: الدوال (Functions)

الدوال هي كتل من الكود يمكن استدعاؤها لتنفيذ مهمة محددة. تساعد في تنظيم الكود وإعادة استخدامه.

شرح مفصل:
  • تعريف الدوال: يتم تعريف الدوال باستخدام `def`.
  • استدعاء الدوال: يمكن استدعاء الدوال باستخدام اسمها.
مثال:
def greet(name) puts "مرحبًا، #{name}!" end greet("علي") # ✅ مرحبًا، علي!

الدرس الثامن: المصفوفات (Arrays)

المصفوفات هي هياكل بيانات تخزن مجموعة من العناصر في ترتيب معين. يمكن الوصول إلى العناصر باستخدام الفهرس.

شرح مفصل:
  • إنشاء المصفوفات: يتم إنشاء المصفوفات باستخدام الأقواس المربعة `[]`.
  • الوصول إلى العناصر: يمكن الوصول إلى العناصر باستخدام الفهرس.
مثال:
numbers = [1, 2, 3, 4, 5] puts numbers[0] # ✅ 1

الدرس التاسع: الهاش (Hashes)

الهاش هو هيكل بيانات يخزن البيانات في أزواج مفتاح-قيمة. يمكن استخدامه لتخزين البيانات المرتبطة.

شرح مفصل:
  • إنشاء الهاش: يتم إنشاء الهاش باستخدام الأقواس المعقوفة `{}`.
  • الوصول إلى القيم: يمكن الوصول إلى القيم باستخدام المفتاح.
مثال:
person = { name: "علي", age: 25 } puts person[:name] # ✅ علي

الدرس العاشر: الاستثناءات (Exceptions)

الاستثناءات هي أخطاء تحدث أثناء تنفيذ البرنامج. يمكن التعامل معها باستخدام `begin`، `rescue`، و `ensure`.

شرح مفصل:
  • التعامل مع الاستثناءات: يتم استخدام `begin` و `rescue` للتعامل مع الأخطاء.
  • الكود النهائي: يتم استخدام `ensure` لتنفيذ كود بغض النظر عن حدوث استثناء أم لا.
مثال:
begin result = 10 / 0 rescue ZeroDivisionError puts "لا يمكن القسمة على صفر!" ensure puts "هذا الكود سيتم تنفيذه دائمًا." end
