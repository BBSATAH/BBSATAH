شرح دورة Objective-C

Objective-C هي لغة برمجة كائنية التوجه تُستخدم لتطوير تطبيقات macOS و iOS. تعتمد على لغة C مع إضافة مفاهيم البرمجة الكائنية. هذه الدورة ستأخذك في رحلة لتعلم أساسيات Objective-C مع أمثلة عملية.

الدرس الأول: مقدمة إلى Objective-C

Objective-C هي لغة برمجة كائنية التوجه تعتمد على لغة C. تُستخدم بشكل رئيسي لتطوير تطبيقات Apple.

شرح مفصل:
  • الفئات (Classes): يتم تعريف الفئات باستخدام `@interface` و `@implementation`.
  • الكائنات (Objects): يتم إنشاؤها باستخدام `alloc` و `init`.
  • الرسائل (Messages): يتم استدعاء الوظائف باستخدام الأقواس المربعة `[]`.
مثال:
#import @interface Person : NSObject @property (nonatomic, strong) NSString *name; @property (nonatomic, assign) NSInteger age; - (void)introduce; @end @implementation Person - (void)introduce { NSLog(@"Hello, my name is %@ and I am %ld years old.", self.name, (long)self.age); } @end // استخدام الفئة Person *person = [[Person alloc] init]; person.name = @"Ali"; person.age = 25; [person introduce];

الدرس الثاني: الخصائص (Properties)

الخصائص هي متغيرات يتم تعريفها داخل الفئة. يمكن التحكم في الوصول إليها باستخدام `@property`.

شرح مفصل:
  • nonatomic: يسمح بالوصول إلى الخاصية من أكثر من thread.
  • strong: يحتفظ بالكائن في الذاكرة حتى يتم تحريره.
  • weak: لا يحتفظ بالكائن في الذاكرة إذا لم يكن هناك مراجع أخرى له.
مثال:
@interface Car : NSObject @property (nonatomic, strong) NSString *brand; @property (nonatomic, weak) id delegate; @end

الدرس الثالث: البروتوكولات (Protocols)

البروتوكولات تشبه الواجهات (Interfaces) في لغات أخرى. يتم استخدامها لتحديد مجموعة من الوظائف التي يجب أن تطبقها الفئة.

شرح مفصل:
  • تطبيق البروتوكول: يتم استخدام `` لتطبيق البروتوكول.
  • الوظائف الاختيارية: يمكن تعريف وظائف اختيارية باستخدام `@optional`.
مثال:
@protocol Printable - (void)print; @end @interface Document : NSObject @end @implementation Document - (void)print { NSLog(@"Printing document..."); } @end

الدرس الرابع: إدارة الذاكرة (Memory Management)

في Objective-C، يتم استخدام **ARC (Automatic Reference Counting)** لإدارة الذاكرة تلقائيًا.

شرح مفصل:
  • ARC: يقوم بتحرير الذاكرة تلقائيًا عندما لا يكون هناك مراجع للكائن.
  • retain و release: تستخدم في إدارة الذاكرة يدويًا (بدون ARC).
مثال:
// مع ARC NSString *name = [[NSString alloc] initWithString:@"Ali"]; // لا حاجة لـ release // بدون ARC NSString *name = [[NSString alloc] initWithString:@"Ali"]; [name release]; // تحرير الذاكرة يدويًا

الدرس الخامس: الكتل (Blocks)

الكتل (Blocks) هي وظائف مجهولة يمكن تمريرها كمعاملات أو تخزينها في متغيرات.

شرح مفصل:
  • بناء الكتلة: يتم تعريف الكتلة باستخدام `^`.
  • تمرير الكتلة: يمكن تمرير الكتلة كمعامل لوظيفة أخرى.
مثال:
void (^simpleBlock)(void) = ^{ NSLog(@"This is a block"); }; simpleBlock(); // This is a block
