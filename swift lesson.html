شرح دورة Swift من الصفر إلى الاحتراف

لتعلم لغة Swift من الصفر إلى الاحتراف، يمكن تقسيم عملية التعلم إلى عدة مراحل متتابعة. إليك خطة دراسة شاملة تبدأ من الأساسيات وتصل إلى المفاهيم المتقدمة.

الدرس الأول: تثبيت Swift والبدء

شرح مفصل:
  • 1. تثبيت Xcode: Xcode هو بيئة تطوير متكاملة (IDE) الرسمية من Apple، ويشمل أدوات لتطوير التطبيقات باستخدام Swift. ستحتاج إلى Xcode لتطوير تطبيقات iOS و macOS باستخدام Swift.
  • 2. تثبيت Swift عبر سطر الأوامر: إذا كنت ترغب في العمل مع Swift دون الحاجة إلى Xcode، يمكنك تثبيتها عبر سطر الأوامر (Command Line).
مثال:
import UIKit print("Hello, Swift!")
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

الدرس الثاني: المتغيرات والأنواع

شرح مفصل:
  • 1. المتغيرات: في Swift، تُستخدم المتغيرات لتخزين البيانات التي قد تتغير أثناء تنفيذ البرنامج. تعرف المتغيرات باستخدام الكلمة المفتاحية `var`.
  • 2. الثوابت: تُستخدم الثوابت لتخزين البيانات التي لا تتغير أثناء تنفيذ البرنامج. تعرف الثوابت باستخدام الكلمة المفتاحية `let`.
مثال:
var name = "Ahmed" name = "Ali" // يمكن تغيير قيمة المتغير print(name) // يطبع "Ali"
let age = 30 // age = 35 // هذا سيتسبب في حدوث خطأ لأن `age` هو ثابت

الدرس الثالث: الثوابت في Swift

شرح مفصل:
  • 1. الثوابت: في Swift، الثوابت هي متغيرات لا يمكن تغيير قيمتها بعد تعيينها. تُستخدم الثوابت عند الحاجة إلى تخزين قيمة لا تتغير طوال فترة تشغيل البرنامج.
  • 2. أنواع البيانات في الثوابت: يمكنك تعريف الثوابت بأنواع مختلفة من البيانات، مثل الأرقام والنصوص والقيم المنطقية.
مثال:
let greeting = "Hello, World!"
let year: Int = 2025 let temperature: Double = 25.5

الدرس الرابع: أهمية لغة Swift

شرح مفصل:
  • 1. السرعة والأداء: واحدة من أكبر مزايا Swift هي السرعة. لغة Swift تم تصميمها لتحسين أداء التطبيقات بشكل كبير مقارنة باللغات التقليدية مثل Objective-C.
  • 2. سهولة التعلم والاستخدام: Swift مصممة لتكون لغة سهل تعلمها وسهلة القراءة والكتابة. لغة البرمجة تعتمد على النحو البسيط والأنيق، مما يجعلها أكثر سهولة للمبتدئين والمحترفين على حد سواء.

الدرس الخامس: مشروع عملي - تطبيق آلة حاسبة بسيطة

شرح مفصل:
  • الفكرة: تطبيق آلة حاسبة بسيطة: المشروع سيكون تطبيق آلة حاسبة يحتوي على واجهة مستخدم (UI) يمكن للمستخدم من خلالها إجراء العمليات الحسابية الأساسية (الجمع، الطرح، الضرب، القسمة).
مثال:
@IBOutlet weak var displayLabel: UILabel! @IBAction func numberPressed(_ sender: UIButton) { let number = sender.titleLabel?.text displayLabel.text = displayLabel.text! + number! }

الدرس السادس: الجمل الشرطية والحلقات

شرح مفصل:
  • 1. الجمل الشرطية: مثل `if`، `else`، و `else if`.
  • 2. الحلقات: مثل `for`، `while`، و `repeat-while`.
مثال:
let age = 18 if age >= 18 { print("أنت بالغ.") } else { print("أنت قاصر.") }
for i in 0..<5 { print("الرقم: \(i)") }

الدرس السابع: الدوال (Functions)

شرح مفصل:
  • 1. تعريف الدوال: يتم تعريف الدوال باستخدام الكلمة المفتاحية `func`.
  • 2. استدعاء الدوال: يمكن استدعاء الدوال باستخدام اسمها.
مثال:
func greet(name: String) { print("مرحبًا، \(name)!") } greet(name: "علي") // ✅ مرحبًا، علي!

الدرس الثامن: المصفوفات (Arrays)

شرح مفصل:
  • 1. إنشاء المصفوفات: يتم إنشاء المصفوفات باستخدام الأقواس المربعة `[]`.
  • 2. الوصول إلى العناصر: يمكن الوصول إلى العناصر باستخدام الفهرس.
مثال:
var numbers = [1, 2, 3, 4, 5] print(numbers[0]) // ✅ 1

الدرس التاسع: الهاش (Hashes)

شرح مفصل:
  • 1. إنشاء الهاش: يتم إنشاء الهاش باستخدام الأقواس المعقوفة `{}`.
  • 2. الوصول إلى القيم: يمكن الوصول إلى القيم باستخدام المفتاح.
مثال:
var person = ["name": "علي", "age": "25"] print(person["name"]!) // ✅ علي

الدرس العاشر: الاستثناءات (Exceptions)

شرح مفصل:
  • 1. التعامل مع الاستثناءات: يتم استخدام `try` و `catch` للتعامل مع الأخطاء.
  • 2. الكود النهائي: يتم استخدام `finally` لتنفيذ كود بغض النظر عن حدوث استثناء أم لا.
مثال:
do { let result = try someThrowingFunction() print(result) } catch { print("حدث خطأ: \(error)") }
